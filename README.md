## 🚀 一、Markdown 基础语法 (入门必学)

Markdown 的核心是使用**简单符号**来代替复杂的格式设置。

### 1. 标题 (Headings)

使用 `#` 符号来表示标题。`#` 的数量越多，标题级别越低（字体越小）。

| 语法 | 渲染效果示例 | 作用 |
| :--- | :--- | :--- |
| `# 一级标题` | **最大标题** | 相当于 HTML 的 `<h1>` |
| `## 二级标题` | **次一级标题** | 相当于 HTML 的 `<h2>` |
| `### 三级标题` | **更小标题** | 相当于 HTML 的 `<h3>` |

### 2. 文本样式 (Emphasis)

用于强调重要内容或术语。

| 语法 | 渲染效果 | 作用 |
| :--- | :--- | :--- |
| `**这是粗体文字**` | **这是粗体文字** | 强调 |
| `*这是斜体文字*` | *这是斜体文字* | 强调或术语 |
| `***这是粗斜体***` | ***这是粗斜体*** | 强烈强调 |
