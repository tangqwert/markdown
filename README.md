## 🚀 一、Markdown 基础语法 (入门必学)

Markdown 的核心是使用**简单符号**来代替复杂的格式设置。

### 1. 标题 (Headings)

使用 `#` 符号来表示标题。`#` 的数量越多，标题级别越低（字体越小）。

| 语法 | 渲染效果示例 | 作用 |
| :--- | :--- | :--- |
| `# 一级标题` | **最大标题** | 相当于 HTML 的 `<h1>` |
| `## 二级标题` | **次一级标题** | 相当于 HTML 的 `<h2>` |
| `### 三级标题` | **更小标题** | 相当于 HTML 的 `<h3>` |

### 2. 文本样式 (Emphasis)

用于强调重要内容或术语。

| 语法 | 渲染效果 | 作用 |
| :--- | :--- | :--- |
| `**这是粗体文字**` | **这是粗体文字** | 强调 |
| `*这是斜体文字*` | *这是斜体文字* | 强调或术语 |
| `***这是粗斜体***` | ***这是粗斜体*** | 强烈强调 |

### 3. 列表 (Lists)

#### 🔸 无序列表

使用 `*`、`+` 或 `-` 后面紧跟一个空格。

```markdown
* 列表项 A
* 列表项 B
+ 嵌套项 1
+ 嵌套项 2

#### 🔸 有序列表

使用数字后面跟一个 . 和一个空格
1. 第一个步骤
2. 第二个步骤
3. 第三个步骤


### 4. 链接与图片
| 类型 | 语法结构 | 实例 |
| :--- | :--- | :--- |
| `链接` | [链接文本](网址) | 点击这里访问 Google |
| `图片` | ![图片描述文字](图片网址或本地路径) | ![一朵花](https://example.com/flower.jpg) |

使用三个或更多的 *、- 或 _ 独占一行。

### 8.
1. 表格(Tables)
虽然表格的语法稍微复杂一点，但非常实用！用 | 来分隔单元格，用 - 来分隔表头和内容。

| 标题 1 | 标题 2 | 标题 3 |
| :--- | :---: | ---: |
| 左对齐内容 | 居中内容 | 右对齐内容 |
| Row 2 Col 1 | Row 2 Col 2 | Row 2 Col 3 |

对齐方式说明：

    :---：左对齐

    :---:：居中对齐

    ---:：右对齐

2. 待办事项列表 (Task Lists)
在列表前加上 [ ] 或 [x] 可以创建可以勾选的列表。

* [x] 完成第一项任务
* [ ] 待办第二项任务
* [ ] 记得买牛奶
